My Portfolio

Smart Key

It is very common to find ourselves in a situation where we have misplaced small items which are hard to trace such as keys, wallets, etc. The main aim of this project is to design and build a working gadget that helps tracing the misplaced keys. This will be achieved by selecting most convenient wireless communication method to help design a gadget to be attached with the key. Design an Arduino based key finder using Bluetooth, GSM and GPS modules. To design a mobile application that will help in locating a key. Test the gadget built and verify if it works.
Tools and Technologies:Android IDE, Arduino Nano, Bluetooth HC-05, LED, Buzzer, Resistors, Jumper wires, SIM 808(GPS/GSM)

Brisk Recognition

“Brisk Recognition” is a effective platform that provide organization to maintain the attendance of employees, students, workers or labors without manual attendance. Organization get daily attendance report in format of .xls file and in tabular format. Organization can add their new staff member or new students using user add feature. Add member using their id, name and face images (frontal and profile face). Recognize staff using automatic attendance. Using this feature one member or multiple members can fill their attendance simultaneously.
Tools and Technologies:IDLE (Python3.7 32 bit),Numpy, Pandas, IP camera, Raspberry pi

Farming Cycle

"Farming cycle" is an app for farmers and vendors. The main aim of this app is to provide a platform for renting machines, selling products, providing information on farm products. It also has a features to scan a leaft of a plant and detecting disease from leaves of plants through image processing using Matlab. So, it will encourage farmers into technical fields with the purpose of making a system which provides diseases detection by using image processing. For vendors side this application provides a way to buy products from farmers and giving farming equipments on rent to the farmers by uploading pictures of their equipments. A tab to view details for growing a crop.
Tools and Technologies:Android IDE, Matlab
